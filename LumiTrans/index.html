LumiTrans: a computer program to fit luminescence decay curves

  • LumiTrans is a computer program written in Java than can fit experimentally obtained luminescence decay curves using a single transfer, cooperative or accretive model. It can account for dipole-dipole (r-6), dipole-quadrupole (r-8) and quadrupole-quadrupole (r-10) interactions.

  • The model to calculate luminescence decay curves is described in detail in:

    • Quantum cutting by cooperative energy transfer in YbxY(1-x)PO4:Tb3+
      P. Vergeer, T.J.H. Vlugt, M.H.F. Kox, M.I. den Hertog, J.P.J.M. van der Eerden, A. Meijerink
      Phys. Rev. B., 2005, 71, 014119 PDF.
