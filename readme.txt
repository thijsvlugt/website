It is possible to have your own personal homepage.
Create your website In this www folder starting with an index.html
Websites with scripting (php, asp) are not possible.
Your website will be accessible from anywhere using this link in a webbrowser:
http://homepage.tudelft.nl/v9k6y

Regulations with respect to use of this facility can be found on 
http://www.tudelft.nl/regulationshomepage

Please do not delete this folder. You will not be able to create a 
personal webpage if this folder is deleted and manually recreated later.
